This is a page dedicated to showing sites that I was a client, Lead Dev, or Junior Dev for.

CLIENT SITE

The Site I was a client for was a website that would teach somebody how to bench press. I found some sample images online that I felt helped readers actualy see what was being taught. I attempeted communicating with the developer through Microsoft teams but I was never really answered. I have no idea how the site turned out I asked for the URL and still havent seen it.

LEAD DEV SITE

The Site I was lead developer for was a site about how to pick up an apple. I managed the development team by giving each Junior dev a page to work on and I made the home page and a universal stylesheet that would be used for all pages. I communicated with the client through Microsoft teams and made small changes as they saw nessacary. I published the clients site on Github and added the junior devs to the repository so we could all work on the site. Managing the site with github was realatively easy and straighforward. REFLECTIONS

The most challenging part of this project was trying to do the junior developer tasks when the lead developer wouldnt respond or create a stylesheet. I feel like I wasnt really able to overcome it as the lead devs seemed to just give up on the project as a whole. I do beleive I did do what was required of me. I enjoyed being lead dev and being able to make a site for an idea somebody else had and to be able to show them that site and have them be excited about it. If I could do the project again I think I would try and update my team on what was required of them and if I was a junior dev I would consistently reach out until I got answers from the lead dev.I think I do feel more confident working on real life projects just beacuse i learnd how the GIT ecosystem kind of works and now it isnt as concerning as I know how it works now.
